# Usage
Install the latest version of NodeJS.

Clone the repo

```
git clone "https://github.com/IainWinter/ChRIS.git" --recurse-submodule
```

Enter the folder

```
cd ChRIS
```

install the package in the existing node_modules directory

```
npm install
```

run the project

```
npm start
```
